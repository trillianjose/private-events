# Private Events

A project that allows user to create events. A user can enroll in a event and can see the number of participants. 

## Tools

* Ruby
* Ruby on Rails

## Instructions

1. `Git clone this repository.`
2. `git checkout dev`
3. `bundle install`
4. `rails db:migrate`
5. `rails server`
